## Description
This Django application offers a solution for managing business operations with an emphasis on user experience and modern web technologies. It integrates Html, Css, Bootstrap, JavaScript for front-end design. The application features models for Buyers, Supplier, Seasons, Categories, Products, Orders, Deliveries, Reports, Stock Management and inventory management.

## Prerequisites
- *Python installed*: Ensure Python is installed on your system. You can download it from the official [Python website](https://www.python.org/).
- *Understand Python and Django*: Basic understanding of Python programming and familiarity with Django web framework.

## Installation

Follow these steps to install the necessary dependencies and set up the application:

### Clone the Repository

bash
git clone https://github.com/AyaanAhmad-code/Inventory-Management-System.git<br>
cd Inventory-Management-System

### Without Docker

#### On Linux

1. *Set Up the Virtual Environment*

    bash
    python3 -m venv venv
    source venv/bin/activate
    

2. *Install Dependencies*

    bash
    pip install -r requirements.txt
    

3. *Apply Migrations and Run the Server*

    bash
    python manage.py migrate
    python manage.py runserver
    

#### On Windows

1. *Set Up the Virtual Environment*

    bash
    python -m venv venv
    venv\Scripts\activate
    

2. *Install Dependencies*

    bash
    pip install -r requirements.txt
    

3. *Apply Migrations and Run the Server*

    bash
    python manage.py migrate
    python manage.py runserver
